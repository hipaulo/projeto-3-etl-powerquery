## 📊 Sobre o projeto
Este dashboard foi desenvolvido durante o curso **"Power BI Desktop: realizando ETL no Power Query"** da Alura.

**Foco principal:** Aprender a **limpar, transformar e preparar dados** antes da análise - a etapa mais importante de qualquer projeto de BI.

## 🔧 Principais técnicas de ETL aplicadas
- Conexão com diferentes fontes de dados
- Limpeza de colunas e remoção de duplicatas
- Tratamento de valores nulos e erros
- Mescla e anexação de tabelas (Merge/Append)
- Criação de colunas condicionais
- Agrupamento e agregação de dados
